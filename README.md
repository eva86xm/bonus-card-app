# Bonus Card App

Прототип клиентского приложения для бонусных карт.

## Что умеет

- вход клиента по телефону;
- отображение бонусной карты;
- отображение баланса бонусов;
- отображение доступных к списанию бонусов;
- отображение истории операций;
- админская привязка существующей карты к телефону.

## Тестовый вход

Телефон:

+7 900 111-22-33

## Файлы

- index.html — структура страницы;
- styles.css — оформление;
- app.js — логика интерфейса;
- api.js — временная заглушка API СНК.

## Что заменить после получения API СНК

В файле api.js нужно заменить тестовые методы на реальные запросы к API СНК:

- findClientByPhone;
- bindCardToPhone;
- получение баланса;
- получение истории операций;
- получение статуса карты.
